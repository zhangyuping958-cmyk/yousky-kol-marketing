# Yousky 手持清洁机 · 海外 KOL 营销拆解

Yousky 手持清洁机品牌出海的 KOL（Key Opinion Leader）营销系统性拆解方案。

## 目录结构

```
├── index.html                          # GitHub Pages 落地页
├── README.md                           # 本文件
└── Yousky手持清洁机海外KOL营销拆解.pdf  # 原版营销拆解文档
```

## 内容模块

| 模块 | 说明 |
|------|------|
| **向内盘点** | KOL 运营专员 / 亚马逊运营 / 产品经理三部门能力与资产核查 |
| **产品与市场** | 竞品拆解、痛点场景化、#CleanTok 爆款逻辑分析 |
| **KOL 金字塔矩阵** | 头部品牌背书 → 腰部垂直带货 → 尾部纯佣铺量 |
| **内容 SOP** | Brief 标准化模板、Before/After 视觉规范、爆款公式 |
| **激励与分佣** | Flat fee / Fee+Commission / CPS 三种模式对比 |
| **数据复盘** | CPM / ROI 追踪看板、Spark Ads 投流放大 |
