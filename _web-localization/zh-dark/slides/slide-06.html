Viewer Actions

从后端反控 前端画布

连接是双向的。不仅视图能向 AI 发送上下文，AI 也可以通过代码执行，直接触发 React 界面中的各种动作。

智能体的工具调用

import { invokeViewerAction } from "pneuma";

await invokeViewerAction({
  id: "navigate-to",
  params: { slide: 10 }
});

Vite 前端立刻响应

视口瞬间跳转至 Slide 10

用户甚至不需要自己滚动页面。当 AI 编辑完第 10 页，它会通过 ViewerContract 直接将你的视线精确带到它刚刚完成工作的地方。
