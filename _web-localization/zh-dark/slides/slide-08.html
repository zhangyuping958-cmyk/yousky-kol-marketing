Design System

基于 CSS 变量 的全局主题

Slide 模式秉持“主题优先”的设计哲学。只需让 AI 或自己修改一个核心的 theme.css，你就能一秒重塑整套幻灯片的视觉美学，绝无死角。

theme.css 
:root {
  --color-bg: #FAF7F2;
  --color-text: #2D2A26;
  --color-primary: #D4632A;
  --color-accent: #C4963A;
  --color-surface: #F0EBE3;
  
  --font-sans: 'Inter', sans-serif;
}
Prompting Tip #1 / 提示词技巧之一

“把全局主题换成最硬核的赛博朋克风。背景全部用极深的反差黑，并用高饱和的霓虹粉和青色做重点元素的点缀。”

Prompting Tip #2 / 提示词技巧之二

“把所有卡片和模块的全局圆角调大，让整套 PPT 给人的感觉更平易近人、更活泼圆润一点。”
