Mode Maker: 让 AI 创造 AI

因为 Pneuma 的每一个功能都被高度抽象在 ModeProtocol 之后，所以我们甚至构建了一个专门用来创造 Mode 的 Mode。

1. 分支与脚手架搭建

启动 mode-maker 模式。告诉 Claude 你想要一个“看板模式”。它会自动拉取基础模板并为你生成所有的 React 初始文件。

2. 实时边写边测

点击“Play”。Pneuma 会嵌套启动一个子服务器来运行你刚写好的 Viewer。在 Claude 飞速写代码的同时，你可以左右分屏实时测试你的应用布局。

3. 一键发布全网

一切就绪后，执行 pneuma mode publish。你的 Mode 会被无缝上传至 R2 节点，全球的 Pneuma 扩展市场将即刻查收到你的作品。
