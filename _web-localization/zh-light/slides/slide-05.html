Capabilities

视图上下文与 智能体视觉

前端视图绝非一个单向的显示器；它能主动与 AI 进行双向交互，赋予智能体对已渲染文档的结构化空间感知能力。

元素精准选中

点击实时预览中的任何区块。Pneuma 会直接用高光将其框选，并自动在你的下一个对话消息中静默注入一段精确的 <viewer-context> 标签。

已选中: <h1>
“把这个标题改大一点，并且换成蓝色”

AI 极其清楚你刚刚点击并选中的是哪一个 <h1>，让你彻底告别费时费力的位置描述。

Agent 可调用动作

ViewerContract 标准合约向外暴露了特定接口，让 Agent 能够直接控制前端 UI 并触发动作，实现惊艳的无缝多步工作流。

大屏导航控制
Agent 在后台编辑修改后，自动强制将你的屏幕翻页到 Slide 10 给你审阅。
视图模式控制
Agent 主动触发切换至“网格视图”，以便从宏观视角整理所有幻灯片。
工作区编排
Agent 根据配置指令完整重置并生成当前工程的骨架结构（危险操作会触发 UI 确认弹窗）。
